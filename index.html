Step-by-Step Presentation

Step 1: Login

Login into AWS console and search for S3 in search Bar.

Step 2: S3 Bucket

Click on create bucket to create one.

Step 3: Create a Bucket

Set a name for your bucket and then make it public accessable.

Step 4: Upload Project

Upload the project to the S3 Bucket and wait till it gets uploaded.

Step 5: Change Properties of Bucket

To acces the project via S3 bucket enable static hosting and also select the index page of your project.

Step 6: Change Policy of Bucket

Add Policy to Bucket to access the content in it.

Step 7: Change Properties of Bucket

To acces the project via S3 bucket enable static hosting and also select the index page of your project.

Step 8: Host Zone a Route 53

Create a host zone for your domain using Route 53 and add records to the host.

Step 9: Change the name server in your Domain

When you create a host zone it will give you a set of name server and the create records for Rule A.

Step 10: Create Certificate for domain

Go to Amazon Certificate Manager and register for certificate to get secure website.

Step 11: Create a Route 53 for Cname

When we create a certificate we should make sure that it has record in route 53.

Step 12: Create a Cloud Front Distribution

Create Distribution this will access the S3 bucket via Cloud Front.

Step 13: Modify the A rule

Once the Distribution is created it will generate a random URL to change that go to route 53 and change the A rule to Cloud front.
